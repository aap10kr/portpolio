KATCH

PLATFORM

Katch는 홈페이지를 만들고 싶지만 어려워 망설이고 있는 분들을 위해 15년 이상의 경험과 노하우가 담긴 Katch만의 아주 특별한 서비스 입니다.
코딩은 할 수 있지만 디자인에 자신이 없는 분들도 Katch에서 마음에 드는 디자인을 구매하여 원하는 대로 수정해 내가 원하는 고퀄리티의 홈페이지를 만들 수 있습니다.
저희 Katch는 일반 기업부터 펜션, 병원, 렌트카 또는 맞춤 디자인 상세페이지, 모션그래픽, 다양한 플랫폼 제작까지 모두 준비가 되어있습니다.
고객님 께서 원하시는 모든 꿈을 함께 이루어 드리겠습니다.

자사 플랫폼 Katch는 웹에이전시를 겸하는 회사 특성상 많은 거래를 이루는 모션그래픽과 영상작업물, 디자인 작업물과 html 파일의 템플릿
소스들 판매를 위한 쇼핑몰플랫폼 제작에 프론트엔드 개발을 전담하여 제작을 진행했습니다.

html CSS Javascript를 이용해 반응형 퍼블리싱을 했으며 java spring MYSQL로 구축한 서버와 데이터베이스를 이용해 제작을 했습니다.
토스페이먼츠 결제 모듈 부착과 Axios와 tymleaf를 사용해 프론트 작업을 진행 했습니다 작업을 진행하는 데에 있어 가장 가장 의견차가 대립했던 부분은

코인 충전 방식인데 코인을 충전해 상품을 주문할 수 있는 방식인데 이러한 결제 방법으로 인해 코인으로 결제, 직접 결제, 코인 선물 등의 코인 관리
방법에 있어 서로의 의견차가 조금씩 달랐고 이는 회의를 크몽 이라는 플랫폼을 참고하게 되었고 크몽 안에서 이루어지는 서비스 로직을 참고해
코인 결제를 직접 해서 플랫폼 안에서 이루어지는 모든 결제는 코인으로만 가능하도록 정해두고 개발을 이어 나갔습니다.

이 외 사용자가 상품을 구매할 때 사용할 판매를 위한 다수의 템플릿을 제작했습니다.
현재 오픈을 앞두고 마지막 테스트를 진행 중이며 추후 유지보수를 진행하며 프로그램에 새로운 기능을 업데이트할 예정입니다.
